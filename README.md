# Machine Learning & Deep Learning Study Notes

Curated notes and references on core machine learning and deep learning concepts.

## Overview

This repository is a personal, evolving knowledge base collecting notes, references, and practical tips gathered while studying machine learning and deep learning. It spans foundational concepts, useful links, and workflow notes for common tools such as Jupyter and AWS SageMaker.

## Contents

| Item | Description |
|------|-------------|
| `Sagemaker/` | Notes and references for working with AWS SageMaker. |
| `DL_Links` | Curated collection of deep learning resources and links. |
| `Domain vs Range of a function` | Notes clarifying the mathematical concepts of domain and range. |
| `Explore TSV Files in BASH` | Command-line tips for inspecting TSV data files. |
| `Jupyter_NB_Issues` | Common Jupyter Notebook issues and their fixes. |
| `ML_Notes` | General machine learning notes and references. |

## Usage

These notes are intended as a quick-reference resource. Browse the individual files and folders for topic-specific material.
